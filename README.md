# Defence Industry Inventory Management System 

A web-based inventory management system for tracking and managing weapon stockpiles.

---

##  Features

* **Secure Access:** Simple authentication portal to prevent unauthorized entry.
* **Armory Overview:** A drop-down menu to select from registered weapons.
* **Stock Control:**  Increase or decrease stock levels dynamically with immediate database updates.
* **Detailed Analytics:** Dedicated preview card descriptions, categories, and real-time inventory counts.

---

##  Layers of software

* **Backend:** PHP 
* **Database:** MySQL 
* **Frontend:** HTML5
