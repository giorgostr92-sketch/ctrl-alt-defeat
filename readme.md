# Σύστημα Διαχείρισης Αποθεμάτων Αμυντικής Βιομηχανίας

## Εισαγωγή

 Το παρόν έργο αφορά την ανάπτυξη μιας διαδικτυακής εφαρμογής διαχείρισης αποθεμάτων αμυντικής βιομηχανίας. Η ιδέα του συστήματος προήλθε από το βιντεοπαιχνίδι PUBG, από το οποίο αντλήθηκαν οι ονομασίες των όπλων που διαχειρίζεται η εφαρμογή.

Σκοπός της εφαρμογής είναι η καταχώρηση, προβολή και διαχείριση όπλων που βρίσκονται σε μια αποθήκη αμυντικής βιομηχανίας μέσω ενός φιλικού περιβάλλοντος χρήστη.

# Περιγραφή Σεναρίου

Το σενάριο βασίζεται στη διαχείριση αποθέματος όπλων μιας αμυντικής βιομηχανίας. Ο χρήστης της εφαρμογής μπορεί:

- Να προσθέτει νέα όπλα.
- Να ενημερώνει το διαθέσιμο απόθεμα.
- Να διαγράφει υπάρχοντα όπλα.
- Να προβάλλει τις πληροφορίες των όπλων.

Η εφαρμογή λειτουργεί σε τοπικό δίκτυο και επικοινωνεί με τον εξυπηρετητή μέσω HTTP Requests.

---

# Αρχιτεκτονική Εφαρμογής

Η εφαρμογή ακολουθεί την αρχιτεκτονική τριών επιπέδων (Three-Tier Architecture).

## 1. Frontend (Παρουσίαση)

Το επίπεδο παρουσίασης αποτελείται από σελίδες HTML μέσω των οποίων ο χρήστης αλληλεπιδρά με το σύστημα.

Οι βασικές σελίδες της εφαρμογής είναι:

- Σελίδα πρόσβασης (Login)
- Σελίδα επιλογής όπλου
- Σελίδα εισαγωγής δεδομένων
- Σελίδα προβολής δεδομένων

## 2. Backend (Λογική Εφαρμογής)

Το backend υλοποιήθηκε με PHP και είναι υπεύθυνο για:

- Τη λήψη αιτημάτων από τον χρήστη.
- Τη σύνδεση με τη βάση δεδομένων.
- Την εκτέλεση SQL εντολών.
- Τη διαχείριση των αποθεμάτων.

## 3. Βάση Δεδομένων

Η αποθήκευση των δεδομένων πραγματοποιείται σε βάση δεδομένων MariaDB/MySQL μέσω του XAMPP.

# Δικτυακή Αρχιτεκτονική

Η εφαρμογή λειτουργεί σε περιβάλλον Cisco packet tracer.

## Υποδίκτυα

Χρησιμοποιήθηκαν τα δίκτυα:

```text
192.168.2.0
192.168.3.0
192.168.4.0
192.168.5.0
192.168.6.0
```

## Δικτυακές Συσκευές

- 5 Routers
- 3 Switches
- 1 Server
- 2Τερματικοί Υπολογιστές (Clients)

## Δρομολόγηση

Για την επικοινωνία μεταξύ των υποδικτύων χρησιμοποιήθηκε το πρωτόκολλο:

```text
OSPF
```

όπου όλα τα δίκτυα ανήκουν στο:

```text
Area 1
```

Ο εξυπηρετητής βρίσκεται στο δίκτυο:

```text
192.168.3.0
```

ενώ οι τερματικοί υπολογιστές βρίσκονται στα δίκτυα:

```text
192.168.4.0
192.168.6.0
```
